# Cuisine Restaurant Landing Page

A modern restaurant landing page built using **HTML5 and CSS3**.

This project recreates a restaurant website layout including a hero section, food gallery, restaurant story, opening hours, and customer testimonial section.

## 🚀 Features

- Responsive landing page layout
- Hero section with search bar
- Restaurant story section
- Food menu gallery
- Opening hours card
- Customer testimonial
- Modern UI layout using Flexbox and Grid
- Clean typography with Google Fonts

## 🛠️ Built With

- HTML5
- CSS3
- Flexbox
- CSS Grid
- Google Fonts

## 📚 Purpose

This project was built as part of my **front-end development practice** to improve skills in layout design, spacing, and modern website structure.

## 📸 Preview

Restaurant landing page interface with food images, menu cards, and booking information.
